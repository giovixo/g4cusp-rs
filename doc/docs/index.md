# g4cusp-gps

__A GEANT4 code to simulate the CUSP detector__

*Source Model: General Purpose Source (GPS)*

![g4cusp screenshot](images/g4cusp.png)

## Usage

Interactive mode:
`./cusp`

Batch mode:
`./cusp batch.mac`
